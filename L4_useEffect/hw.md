## Backend

для бэка можно использовать json-server.  
установка `yarn global add json-server`  
запуск `json-server --watch PATH/TO/DB --port 3033 `  
пример пути `src/ToDo/server/db.json`  
пример базы данных

```
{
"list": []
}
```
